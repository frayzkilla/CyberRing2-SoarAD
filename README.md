# CyberRing2-SoarAD
Сервис предназначен для автоматизации реагирования на инциденты информационной
безопасности. Платформа позволяет создавать тикеты, назначать исполнителей по ним, выполнять
скрипты реагирования (сбор процессов, изоляция хостов, блокировка учётных записей и т.д.) и
управлять пользователями с разграничением по организациям.
Тех.стек сервиса: Python3, Flask, SQLite3, Jinja2, Docker + Docker-Compose
