# CyberRing2-SoarAD
Сервис предназначен для управления корпоративными проектами и партнерским взаимодействием.
Платформа позволяет создавать публичные и скрытые проекты, вести по ним заметки и файлы,
управлять партнерскими каналами и приглашениями, выдавать гранты доступа к проектам,
восстанавливать доступ к учётным записям и вести личный блокнот, а также управлять пользователями.
Тех.стек сервиса: Python3, Flask, Jinja2, PostgreSQL, Docker + Docker-Compose
